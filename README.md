# ubirch-poc-manager-app
App for management of Points of Certification (PoC)

## Run App

### Prerequisites

* [Node.js](https://nodejs.org/)
* [ionic framework](https://ionicframework.com/getting-started):


        npm install -g ionic


### Start app on localhost

    npm start

Open Web browser on

    http://localhost:9202
